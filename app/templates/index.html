Welcome to the demo of the website created to complete the midterm in the course Advanced Algorithms and Programming. The website is created by Siri Nordlander and Rebecca Fällström.
How to navigate:
HOME -> this page, the homepage
TABLE -> instert a date to see the specific data from that day
SHOW CLOSING PRICES -> pick two dates and get the closing prices inbetween those dates. Pick a number of elements you want to see if you want the list to be shoreter and only see the n:th first elements
GRAPH -> to see a graph of all closing prices during the interval
